# dotfiles

Clone this repository:

```bash
git clone --recurse-submodules https://github.com/tarik2281/dotfiles.git
```

After cloning without submodules:

```bash
git submodule update --init
```

Run init script:

```bash
cd dotfiles
./init.sh
```
